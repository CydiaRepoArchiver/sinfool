Sinfool's Repo

This is Sinfool's Cydia repository! This is part of a long project to turn all of Sinfool's Flex Patches into tweaks.

The scripts, this website, and a few other components of this project were created by ipad_kid. This would not have been possible without the help of John Coates, the developer of Flex, who's really awesome, nice, and funny. Also thank you to Plexigen for the amazing repo icon

I'm working on going through each tweak, adding Settings panels, fixing descriptions, and anything else that may need to be done. Currently these tweaks are generated by a tool that was created specifically for this. Unfortunately there are a few issues, but we're working through them. Thank you, and sorry for any inconveniences. Please report bugs below
