Sinfool's Repo

This is Sinfool's Cydia repository! This is part of a long project to turn all of Sinfool's 520 Flex Patches into tweaks.

The scripts, this website, and a few other components of this project were created by ipad_kid (that's me, hi). This would not have been possible without the help of John Coates, the developer of Flex, who's really awesome, nice, and funny.

At this time there are only 60 tweaks available here, and I haven't really even looked through them yet. Over the next two weeks, all five hundred will hopefully be available here, and then I'll begin slowly going through each one adding Settings panels, fixing descriptions, and anything else that may need to be done. Currently these tweaks are generated by a script that was created specifically for this, unfortunately there are a few issues, but we're working through them. Thank you, and sorry for any inconveniences
